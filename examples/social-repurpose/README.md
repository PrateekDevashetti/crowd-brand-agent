# social-repurpose

One topic → consistent images for blog/X (16:9), Instagram feed (1:1), and
Stories (9:16). Uses generative resize from a single hero so the campaign
looks like one campaign.

```bash
BRAND_SESSION_ID=<id> node index.mjs "We just launched dark mode"
```

Wire the printed URLs into your scheduler (Buffer, n8n, Zapier webhook, or a
plain cron job).
