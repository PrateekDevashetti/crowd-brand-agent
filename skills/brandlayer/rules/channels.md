# Channel → aspect ratio

| Channel | Use | aspectRatio |
| --- | --- | --- |
| Instagram feed | post | 1:1 |
| Instagram story / Reels | vertical | 9:16 |
| LinkedIn post | feed image | 1:1 |
| X / Twitter | inline image | 16:9 |
| YouTube thumbnail | preview | 16:9 |
| Pinterest pin | tall pin | 9:16 |
| Blog hero | article header | 16:9 |
| Email header | newsletter banner | 16:9 |
| Display ad (leaderboard-ish) | web banner | 16:9 |
| Presentation cover | slide | 16:9 |
| Print poster | portrait | 3:4 |

Resolution: default `2K` (1 credit). Use `4K` (2 credits) only when the user
asks for print or large-format output. `variants` 2–4 when the user wants
options to choose from.

Note: with the OpenAI provider, ratios map to the nearest of 1:1 / 3:2 / 2:3.
